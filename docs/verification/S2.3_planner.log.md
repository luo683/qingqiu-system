## 5. 设计要点

### 复用
- 复用现有 Step + Plan dataclass + 3 规则模板 + LLM 拆解
- 不引入新依赖（标准库实现 Kahn 算法 + 拓扑）

### 算法
- **topological_sort**: Kahn 算法（in-degree + queue）— O(V+E)
- **parallel_groups**: 计算每个 step 的深度（最长依赖链），按深度分组
- **cycle detection**: Kahn 算法未处理完所有节点 → 剩余即环
- **mermaid**: 标准 graph TD 格式，兼容 GitHub README 渲染

### 兼容
- to_dict() 保持原有格式（不破坏现有调用者）
- topological_sort() 报错 raise ValueError（fail-fast）
- parallel_groups() 返回空 list（边界 case 安全）
