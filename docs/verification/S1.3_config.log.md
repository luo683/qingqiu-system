## 测试实现细节

- 轮询间隔：500ms（验收 1s 内生效有充足余量）
- 文件签名检测：`(mtime, size)` 元组（避免 Windows mtime 精度问题）
- 签名初始化：**在 `start_watching` 同步初始化**（避免 task 启动时读到"修改后"签名）
- 加载失败兜底：try/except + 保留旧 Config 对象（不让损坏的 config 拖垮整个 daemon）
