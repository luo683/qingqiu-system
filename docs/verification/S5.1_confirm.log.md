## 5. 设计要点

- **Prompter 抽象类**：未来 Voice（pygame/tts）/ IM（飞书）实现只需 `class VoicePrompter(Prompter)` 即可
- **CLIPrompter 后台线程 + join(timeout)**：超时返回 False 不阻塞主流程
- **ConfirmTimeout 继承 ConfirmRejected**：调用方只需 catch ConfirmRejected（简化调用代码）
- **Confirm 单例 + ask() 函数**：日常使用一行 `ask("...")` 即可
- **default 60s timeout**：符合 PRD §6.2 "60 秒不响应 → 自动拒绝"
