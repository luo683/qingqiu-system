## 1. 文件

```
src-tauri/
├── Cargo.toml          · Rust 包配置（tauri 2 + tray-icon + shell plugin）
├── tauri.conf.json     · 应用配置（窗口 + 托盘 + bundle）

    └── main.rs         · 入口（系统托盘 + webview → FastAPI 7788）
```

## 2. 关键设计

- **集成 FastAPI**：Tauri webview 加载 `http://127.0.0.1:7788`（已有的 daemon）
- **系统托盘**：左键单击 → 显示主窗口；右键菜单 → "显示主窗口" / "退出"
- **单实例**：默认 Tauri 行为
- **打包**：`tauri build`（3 个平台：.msi/.dmg/.deb/.appimage）

## 3. 不做的原因（用户决策）

- 3 天估时 + Rust 编译环境 + Icon 资源（4 个尺寸）
- M9 web UI 已 work，等价 UX（用浏览器开 `http://127.0.0.1:7788`）
- **价值只有 1**：用户双击图标启动（vs 现在命令行 `qingqiu start`）
- **收益不抵成本**

## 4. 启用方式（如要做）

```bash
# 1. 装 Rust 工具链（一次性，~5 分钟）
rustup-init.exe

# 2. 装 Tauri CLI
cargo install tauri-cli --version "^2.0"

# 3. dev 模式（带热重载）
cd src-tauri && cargo tauri dev

# 4. 打包
cd src-tauri && cargo tauri build
# → src-tauri/target/release/bundle/{msi,dmg,deb,appimage}/
```
