## 6. Piper 使用说明（用户挂载）

如需启用 Piper（高保真神经语音）：

```python
from qingqiu.voice.tts import PiperTTS, get_default_backend

# 1. 装 piper-tts: uv add piper-tts
# 2. 下载模型：https://github.com/rhasspy/piper/releases
#    → zh_CN-huayan-medium.onnx (60MB)
# 3. 实例化
piper = PiperTTS(model_path=Path("models/zh_CN-huayan-medium.onnx"))
print(piper.is_available())  # True

# 或全局替换 backend
piper.speak("你好清秋", out_path="out.wav")
```
