## 5. 设计要点

- **Intent 枚举**：18 个意图（基于现有 CLI 子命令），`from_str()` 容错
- **RuleBasedClassifier**：regex 模式 + 中文友好的 lookbehind/ahead
- **LLMClassifier**：异步调用 LLM（OpenAI/Anthropic/Ollama/Custom 任一），要求返回 JSON
- **IntentClassifier 主类**：规则 → LLM → fallback 三层
- **ClassificationResult**：`intent` + `source` + `confidence` + `raw_text` + `reason`
- **中文边界 bug 修复**：用 `(?<![a-zA-Z0-9_])` 替代 `\b`（中文不在 `\w` 里）
- **任务列表支持两种顺序**：`task list` + `看任务` 都识别
