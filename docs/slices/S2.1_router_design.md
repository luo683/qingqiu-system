## 4. 接口设计

### 4.1 全局 flag

```python
parser.add_argument("-V", "--version", action="store_true")
parser.add_argument("-v", "--verbose", action="store_true")
parser.add_argument("--json", action="store_true", help="JSON 输出")
parser.add_argument("--no-color", action="store_true", help="禁用 ANSI")
parser.add_argument("--config", type=Path, help="临时 config 文件")
```

### 4.2 输出格式

```python
# src/qingqiu/cli/output.py
class OutputFormatter:
    def __init__(self, json_mode: bool = False, no_color: bool = False): ...
    def print(self, data: Any, title: str | None = None) -> None: ...
    def error(self, message: str, code: int = 1) -> None: ...
    def table(self, rows: list[dict], columns: list[str]) -> None: ...
```

- **human 模式**（默认）：彩色 + 表头 + emoji 前缀
- **json 模式**（`--json`）：`json.dumps(data, indent=2, ensure_ascii=False)`

### 4.3 错误体系

```python
# src/qingqiu/cli/errors.py
class CLIError(Exception):
    """CLI 错误基类"""
    code: int = 1
    def __init__(self, message: str, hint: str | None = None): ...

class UserError(CLIError):
    code = 1   # 用户错（参数、用法）

class SystemError(CLIError):
    code = 2   # 系统错（IO、网络、数据库）

class NotFoundError(UserError):
    """资源不存在（task/memory key）"""

class ValidationError(UserError):
    """参数校验失败"""
```

统一处理：
```python
# cli/main.py
def main():
    ...
    try:
        dispatch(args)
    except CLIError as e:
        out.error(str(e), code=e.code)
        sys.exit(e.code)
    except KeyboardInterrupt:
        out.error("interrupted", code=130)
        sys.exit(130)
```

### 4.4 子命令 dispatcher

```python
# cli/main.py
SUBCOMMANDS = {
    "ask": (cli.ask, "add_argument_prompt"),
    "chat": (cli.chat, None),
    "task": (cli.task, "add_subparsers_task"),
    ...
}

def dispatch(args) -> int:
    if not args.command:
        parser.print_help()
        return 0
    module, builder = SUBCOMMANDS[args.command]
    return module.run(args)
```

| 风险 | 严重度 | 缓解 |
|------|-------|------|
| cli.py → cli/ 拆分破坏老命令 | 中 | 保留 `src/qingqiu/cli.py` 顶层 wrapper 调 `cli/main.py:main()`；pyproject 入口不变 |
