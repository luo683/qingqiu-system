# DESIGN · 清秋前端设计语言

> **状态：** v0.3.0 · 配套 PRD v0.2.2 / TECH-STACK v0.3.0
> **作者：** Mavis
> **核心约束：** **科技感 + 简约大气 + 没有 AI 味**

---

## 1. 一句话设计哲学

**清秋的视觉应该像 IDE / 终端 / 监控仪表盘的私生子，不是聊天机器人的孙子。**

不是聊天窗口、不是卡片堆叠、不是渐变光晕 —— 是**控制台美学**：
- 深色 + 单一强调色
- 等宽字体打底
- 几何线条优先于装饰
- 信息密度高，反对大块留白居中
- 不拟人化、不卖萌、不问候

---

## 2. 五大原则

| # | 原则 | 含义 | 反例 |
|---|------|------|------|
| D1 | **克制用色** | 单色 / 双色优先，最多三个色阶 | ~~紫渐变 + 蓝紫罗兰~~ |
| D2 | **几何优先** | 直线、栅格、矩形为主，圆角 ≤6px | ~~超大圆角 pill 按钮~~ |
| D3 | **终端化** | 等宽字体、ASCII 边框、命令式 UI | ~~居中大头像 + 大标题~~ |
| D4 | **高密度** | 信息密度优先，留白为功能服务 | ~~空 80% 屏幕放一张插画~~ |
| D5 | **不拟人** | 不问候、不卖萌、不"我在这里帮您" | ~~"你好呀！我是清秋"~~ |

## 3. 配色（深色主题 · v1.0 唯一）

| 名称 | Hex | 用途 |
|------|-----|------|
| `--qq-bg` | `#0A0E14` | 主背景 |
| `--qq-bg-elevated` | `#131820` | 卡片 / 面板 |
| `--qq-bg-overlay` | `#1B212C` | 弹窗 / 浮动层 |
| `--qq-border` | `#222A36` | 默认边框 |
| `--qq-border-strong` | `#3A4452` | hover / focus |
| `--qq-text-primary` | `#E6EDF3` | 主要文字 |
| `--qq-text-secondary` | `#8B949E` | 次要文字 |
| `--qq-text-tertiary` | `#6E7681` | 三级 / placeholder |
| **`--qq-accent`** | **`#00FFD1`** | **唯一强调色 · 冷青** |
| `--qq-accent-dim` | `#00B89C` | 强调 dim 状态 |
| `--qq-accent-bg` | `#0A2E2A` | 强调做背景 |
| `--qq-success` | `#3FB950` | 成功 |
| `--qq-warning` | `#D29922` | 警告 |
| `--qq-danger` | `#F85149` | 危险 |

**为什么冷青：** 不是紫色 / 蓝色（AI 味的标志）；冷色 + 高饱和；类似终端绿光；行业少用。

**禁止：** ❌ 渐变 / ❌ 多色 / ❌ 紫色 / ❌ 阴影装饰

---

## 4. 字体

```css
--qq-font-mono: 'JetBrains Mono', 'Cascadia Code', 'Consolas', monospace;
--qq-font-sans: 'Inter', 'PingFang SC', 'Microsoft YaHei', system-ui, sans-serif;
```

**默认 mono**，仅大标题用 sans。**不用装饰字体**、**不用渐变文字**。

---

## 5. 间距 / 圆角 / 阴影 / 动效

| 维度 | 规则 |
|------|------|
| **间距** | 8px 栅格：4 / 8 / 12 / 16 / 24 / 32 / 48px |
| **圆角** | 按钮 2px / 输入框 2px / 卡片 4px / 弹窗 6px / **不用 pill（999px）** |
| **阴影** | sm 0 1px 2px / md 0 4px 12px / lg 0 12px 32px（都为层级，不装饰） |
| **动效** | hover 80ms / 状态 120ms / 弹窗 160ms / **不用弹跳 / 旋转 / 流光** |

---

## 6. 图标

- **库：** `lucide-react`
- **风格：** outline（线性）
- **尺寸：** 16 / 20 / 24px
- **默认色：** `--qq-text-secondary`，active `--qq-accent`
- ❌ **不用 emoji**

---

## 7. 组件清单

| 组件 | 关键规范 |
|------|---------|
| **Button** | 矩形 + 2px 圆角 + 三态（default/hover/active） |
| **Input** | 单底线 + focus 底线变 accent |
| **Card** | `bg-elevated` + border + 4px 圆角 |
| **Modal** | `bg-overlay` + 6px 圆角 + 蒙层 |
| **Toast** | 右上角 + 矩形 + 4 秒自动消失 |
| **List** | 行高 32px + hover `bg-elevated` |
| **Confirm** | **嵌入对话流**（像 IDE Quick Fix），不用 Modal |

### 不用（**禁止**）

- ❌ Avatar 圆形头像
- ❌ 大圆角 Card
- ❌ Glassmorphism / 毛玻璃
- ❌ Neumorphism / 新拟态
- ❌ 拟物化
- ❌ 插画 / 卡通
- ❌ "Hello, I'm xxx" 开场白
- ❌ Emoji 装饰
- ❌ 落地页大标题大按钮

## 9. 反 AI 味清单（**最重要**）

| # | 不能做 | 替代 |
|---|--------|------|
| 1 | 紫渐变 + 蓝紫罗兰主色 | 冷青 `#00FFD1` |
| 2 | 居中大圆头像 + "你好我是 xxx" | 直接进功能流 |
| 3 | Pill 按钮（圆角 999px） | 矩形 + 2px |
| 4 | "试试问 xxx" prompt suggestions | 不做建议 |
| 5 | 流光 / 渐变 / 发光按钮 | 矩形 + 单色边框 |
| 6 | Emoji 作为图标 / 装饰 | lucide 图标 |
| 7 | Glassmorphism / 毛玻璃 | 实色 + 边框 |
| 8 | 落地页大标题大按钮 | 直接功能 UI |
| 9 | 对话气泡区分用户/助手 | `>` + `←` 等宽前缀 |
| 10 | 拟人化语气（"我帮您"） | 命令式语气 |
